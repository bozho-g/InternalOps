# InternalOps

Internal request management system with role-based authorization and real-time notifications.

## Tech Stack
- **Backend:** ASP\.NET Core, Entity Framework Core, SignalR, Azure SQL
- **Frontend:** React, React Query, Axios
- **Cloud:** Azure Blob Storage

## Architecture
- Service layer pattern with dependency injection
- RESTful API with role-based access control (Admin/Manager/User)
- Real-time SignalR integration for notifications
- EF Core migrations for database versioning
